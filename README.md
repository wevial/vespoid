# Vespoid 🐝

A personal job search co-pilot. Scrapes tailored listings via Playwright, tracks application statuses, single-user.

**Read the full spec:** [PROJECT.md](./PROJECT.md)

## Quick Start

```bash
git clone https://github.com/wevial/vespoid.git
cd vespoid
bun install
bunx convex dev
bun run dev
```
